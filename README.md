# ikedamclock for M5Paper

## Usage

1. Install VSCode.
2. Install platformio in VSCode.
3. Open this folder with VSCode.
4. Connect your M5Paper.
5. Open the PLATFORMIO pain and run "Platform > Upload Filesystem Image" in PROJECT TASKS.
6. Run "General > Build" in PROJECT TASKS.
7. Run "General > Upload" in PROJECT TASKS.
